# SignIn Form



### Installation

1. Navigate to the Project Directory:

    ```bash
    cd "mini-projects(React)/SignIn-page"

2. Install Dependencies

    ```bash
    npm install

3. Start the server

    ```bash
    npm start

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
